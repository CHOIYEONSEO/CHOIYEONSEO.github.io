## 비어있는 list 생성하기
list = list()   

## 한 줄에 입력하는 값들 list로 입력 받고 싶을 때
미리 list 만들 필요 없이   
- 문자열로 입력 받을 때
list = list(input().split())   

- 숫자로 입력 받을 때
list = list(map(int, input().split()))   

## 리스트 정렬하기
list.sort()   
list의 요소가 숫자면 작은수에서 큰수로 정렬해줌.   
list의 요소가 문자면 알파벳 순서대로 정렬해줌.   

## 리스트에 값 추가하기
list.append()

## 리스트에서 값의 인덱스 찾기
list.index(인덱스 궁금한 값)
