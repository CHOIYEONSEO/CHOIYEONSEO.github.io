## for i in range(a,b,c) 중 i값
c가 양수일때, i값은 a부터 b-1까지 |c|만큼 증가한다.<br>
c가 음수일때, i값은 a부터 b+1까지 |c|만큼 감소한다.<br><br>

예를 들어,
```python
for i in range(3, 0, -1):
  print(i)

## 출력
# 3
# 2
# 1
```

따라서 만약 i가 0일때까지 반복하고 싶다면,
```python
for i in range(3, -1, -1):
  print(i)

## 출력
# 3
# 2
# 1
# 0
```
위와 같이 두번째 인수를 0이 아닌 -1로 적어줘야 됨.

## 1ms
1ms은 0.001초다.
