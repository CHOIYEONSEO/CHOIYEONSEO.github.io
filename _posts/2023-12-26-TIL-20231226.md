## 파이썬 값 입출력
- input() 함수
  : 값을 입력받을 수 있음. Enter로 값 구분. 문자열로 받게됨.

- print() 함수
  : 값 출력하기

## 파이썬 값 한줄에 여러 값 입력받기
###### 방법1
[변수 문자열]   
변수1, 변수2 = input().split()   

[변수 숫자]   
변수1, 변수2 = input().split()   
변수1 = int(변수1)   
변수2 = int(변수2)   

###### 방법2
[변수 숫자]   
변수1, 변수2 = map(int, input().split())   

## 연산자(나누기 관련)
- /    
  : 기본적 나누기 연산자. 실수형(float)으로 리턴    
- //    
  : int(정수형)으로 몫만 리턴.    
- %   
  : 나머지 값 리턴.   
- divmod()    
  : ie. a, b = divmod(2, 3)
    a에 몫 저장, b에 나머지 저장.   

## for 반복문
- 전형적 for 반복문   
  : ie. test_list = ['one', 'two', 'three']   
        for i in test_list:   
          print(i)   
- range와 함께   
  : ie. for i in range(1, 11):   
          print("*"*i)   
    range는 1부터 11-1까지 반복됨. 즉 10번 반복함.
