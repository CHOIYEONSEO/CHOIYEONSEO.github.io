# 용어 정리
- 데이터베이스 : 데이터의 집합. 
- DBMS : Database Management System. 데이터를 잘 관리할 수 있도록 만든 시스템. Oracle, MS-SQL 등이 일종.
- 테이블 = 엔터티(entity)
- 행 = 로우 = 인스턴스
- 열 = 컬럼 = 속성(attribute)

# 모델링이란
## 정의
현실 세계의 비즈니스 프로세스와 데이터 요구 사항을 추상적이고 구조화된 형태로 표현하는 과정

> 데이터를 저장해야 될 상황이 생겼을 때 데이터의 요구사항을 만들어가는 과정이라고 생각하면 된다.
